# Sortes

An Android app for reading academic papers.

## Features

- Browse and organize your PDF paper library
- Clean, distraction-free reading experience
- Annotation and highlighting support
- Search across your library

## Getting Started

1. Clone the repo
2. Open in Android Studio
3. Build and run on a device or emulator (API 26+)
